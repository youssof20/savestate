Copy the folder:

  <repository root>/addons/savestate

into this directory so you have:

  addons/savestate/

Or extract only addons/savestate from savestate-lite-*.zip (GitHub Releases).

Then open this project in Godot and enable SaveState (Lite) under Project Settings -> Plugins.

Contributors: copied addons/savestate/ and addons/savestate_pro/ under this sample are listed in the repo-root .gitignore so the canonical addon folders at the repository root are not duplicated in git.
